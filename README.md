# RockVsMine_prediction

# Objective: 

The main objective of this project is to classify whether the sonar is hitting a rock or a mine.

Dataset: The dataset is from kaggel( SONAR MINE DATASET)
https://www.kaggle.com/datasets/mayurdalvi/sonar-mine-dataset

Methodology: Logistic Regression has been employed as the ML model to solve this classification problem.

# Model Performance:
The model achieved an accuracy of:

Training Accuracy: 83.42%

Testing Accuracy: 72.19%
